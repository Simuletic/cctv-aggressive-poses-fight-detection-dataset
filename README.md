# cctv-aggressive-poses-fight-detection-dataset
CCTV High-quality Aggressive Pose Estimation for Violence &amp; Fight Detection annotated with Yolo bounding boxes. A realistic synthetically generated dataset from https://simuletic.com with no privacy issue.

The full dataset to be found here: https://www.kaggle.com/datasets/simuletic/cctv-aggressive-poses-and-fight-detection-dataset
